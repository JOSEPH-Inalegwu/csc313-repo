Study Note 01

Intelligent Agents & Rationality

An AI Agent perceives its environment through sensors and acts upon it through actuators. It strives to achieve goals by performing actions.

Definition of Rationality

"Acting rationally means doing the right thing, given what the agent knows. The 'right thing' is the action that is expected to maximize the agent's performance measure."

Performance Measure: This is the objective criterion that defines success for the agent.

Study Note 02

Hierarchy of AI Agents

  • Simple Reflex Agents: Act based on current perception only (e.g., thermostat).
  • Model-Based Reflex Agents: Maintain internal state to handle partial observability.
  • Goal-Based Agents: Consider future outcomes to reach a specific goal.
  • Utility-Based Agents: Maximize a "utility" function (happiness/preference).
  • Learning Agents: Improve performance over time from experience.
Study Note 03

Environments: The PEAS Framework

To design an agent, we must specify its task environment using PEAS:

  • Performance: Measure of success.
  • Environment: The world it operates in.
  • Actuators: How it acts on the world.
  • Sensors: How it perceives the world.
Example: Thermostat

P: Energy efficiency/Comfort; E: Room/Building; A: Heater/AC; S: Temperature sensors.

Study Note 04

Problem Formulation & Search

Before searching, an agent must define the State Space (all possible configurations).

  • Initial State: Starting point.
  • Actions: Available moves.
  • Transition Model: Result of an action.
  • Goal Test: Determines if goal is reached.
  • Path Cost: Cost of the sequence of actions.

Example: Navigating from "Angwan Lambu gate" to "Department of Computer Science".

Study Note 05

Uninformed Search Strategies

These explore blindly without domain-specific knowledge:

  • Breadth-First Search (BFS): Explores layer by layer. Complete and Optimal for uniform costs.
  • Depth-First Search (DFS): Explores deep before backtracking. Uses a stack. Not always complete.
  • Uniform Cost Search (UCS): Expands lowest path cost first. Optimal.
Study Note 06

Informed Search & A* Algorithm

Informed search uses Heuristics (h(n)) to estimate distance to the goal.

  • Greedy Best-First: Explores node closest to goal. Can get stuck.
  • A* Search: Combines actual cost (g) and estimated cost (h). f(n) = g(n) + h(n).
A* Optimal Property

A* is both complete and optimal if the heuristic is admissible (never overestimates).
